# Alpine Linux 镜像源使用帮助

## 收录架构

- armhf
- x86
- x86_64

## 使用说明

编辑/etc/apk/repositories，先注释掉里面的所有行，然后在文件的最顶端添加(注意将3.3换成需要的版本)

    Server = http://mirrors.ustc.edu.cn/alpine/v3.3/main/

[参考wiki](http://wiki.alpinelinux.org/wiki/Alpine_Linux_package_management#Packages_and_Repositories)
