使用方法
========

#. 打开 Software Manager ，点击 Edit ⇒ Software Sources ；
#. 在 Download from 中选择 Other ；
#. 选择 China ⇒ mirrors.ustc.edu.cn ；
#. 点击 Choose Server ；
