=========================
Arch Linux ARM 源使用帮助
=========================

地址
====

https://mirrors.ustc.edu.cn/archlinuxarm/

说明
====

Arch Linux ARM 软件源

收录架构
========

ARMv7, AArch64

使用说明
========

编辑 :file:`/etc/pacman.d/mirrorlist` ，在文件的最顶端添加

::

    Server = https://mirrors.ustc.edu.cn/archlinuxarm/$arch/$repo
