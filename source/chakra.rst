=====================
Chakra 源使用帮助
=====================

地址
====

https://mirrors.ustc.edu.cn/chakra/

说明
====

Chakra GNU/Linux 软件源

收录架构
========

x86_64

使用说明
========

编辑 :file:`/etc/pacman.d/mirrorlist` ，先注释掉里面的所有行，然后在文件的最顶端添加 

::

    Server = https://mirrors.ustc.edu.cn/chakra/$repo/$arch
