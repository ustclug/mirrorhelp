# Arch Linux ARM

## 地址

<https://mirrors.ustc.edu.cn/archlinuxarm/>

## 说明

Arch Linux ARM 软件源

## 收录架构

ARMv7, AArch64

## 使用说明

编辑 `/etc/pacman.d/mirrorlist`，在文件的最顶端添加

```ini
Server = https://mirrors.ustc.edu.cn/archlinuxarm/$arch/$repo
```
