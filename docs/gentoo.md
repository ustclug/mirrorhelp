# Gentoo 源使用帮助

## 地址

<https://mirrors.ustc.edu.cn/gentoo/>

## 说明

Gentoo 软件源

## 收录架构

ALL

## 使用说明

在 `/etc/portage/make.conf`{.interpreted-text role="file"}
中添加或修改：

    GENTOO_MIRRORS="https://mirrors.ustc.edu.cn/gentoo/"
